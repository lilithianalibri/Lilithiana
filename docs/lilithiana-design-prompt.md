# Prompt Design WOW - LILITHIANA

Use this prompt with a UI generator (v0, Figma AI, or similar):

```text
Design a premium web experience for "LILITHIANA", an audiobook platform focused on books by women authors.

Goal:
- Build a modern, elegant, emotionally rich interface inspired by Audible UX patterns.
- Feminine tone but sophisticated, not stereotyped, not childish.
- Keep structure production-ready, with realistic content placeholders and reusable components.

Brand Direction:
- Name: LILITHIANA
- Positioning: "Audiolibri di autrici, ascolto con anima."
- Visual mood: editorial luxury + cinematic audio culture.
- Style keywords: refined, immersive, poetic, confident, contemporary.

Color System:
- Base background: warm ivory / pearl.
- Accent 1: deep burgundy.
- Accent 2: rose copper.
- Accent 3: soft gold highlights.
- Text: dark graphite, high contrast.

Typography:
- Display font with personality for titles (editorial feel).
- Clean sans-serif for body and UI.
- Strong hierarchy with generous spacing.

Required Pages / Sections:
1) Homepage:
   - Hero with bold statement, CTA "Esplora catalogo", CTA "Ascolta ora".
   - Featured audiobook module with progress/resume state.
   - Grid catalog of fake audiobooks (cover, title, author, narrator, duration).
   - Section explaining chapter organization model.
   - "How listening works" 3-step UX explanation.
2) Book Detail Page:
   - Large cover.
   - Title, author, narrator, description, total duration.
   - Primary button: "Ascolta ora".
   - Resume listening card.
   - Ordered chapter list:
     Prologo/Introduzione
     Capitolo 1
     Capitolo 2
     Capitolo 3
     Epilogo
     Eventuali contenuti extra
   - Support split chapters: "Capitolo X - Parte 1", "Capitolo X - Parte 2".

UX Rules:
- Chapters are internal to one audiobook, never separate catalog items.
- User enters a book, presses play, continues from last point or starts from beginning.
- Chapter list is used for navigation and quick jumps.

Motion:
- Add meaningful animations: hero reveal, card stagger, soft hover lifts.
- Avoid noisy animation; elegant and smooth transitions only.

Visual Craft:
- Use textured gradients, soft light blooms, and glass-like panels.
- Use refined iconography for audio/listening metadata.
- Build desktop and mobile responsive layouts.

Deliverables:
- High-fidelity mockup feel.
- Componentized structure ready to implement in Next.js + Tailwind.
- Include realistic fake content for at least 4 audiobooks with chapter lists.
```
